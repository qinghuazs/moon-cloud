使用帮助

快速上手 Moon Cloud 短链服务

快速开始

1. 输入原始链接

在首页输入框中粘贴您要缩短的长链接， 支持 HTTP 和 HTTPS 协议。

2. 自定义短链（可选）

您可以自定义短链标识符， 让链接更有意义和易记。

3. 生成短链

点击"生成短链"按钮， 系统将立即为您创建短链接。

4. 复制分享

复制生成的短链接， 即可在任何地方使用和分享。

API 接口

生成短链
POST /api/generate

请求体示例：

{ "originalUrl": "https://www.example.com/very/long/url", "customShortUrl": "mylink", "userId": "user123" }

响应示例：

{ "success": true, "shortUrl": "abc123", "fullShortUrl": "http://localhost:8080/abc123", "originalUrl": "https://www.example.com/very/long/url", "isNew": true, "message": "短链生成成功" }
解析短链
GET /api/resolve/{shortUrl}

响应示例：

{ "success": true, "originalUrl": "https://www.example.com/very/long/url", "shortUrl": "abc123" }
获取统计信息
GET /api/stats/{shortUrl}

响应示例：

{ "success": true, "shortUrl": "abc123", "originalUrl": "https://www.example.com/very/long/url", "clickCount": 156, "status": "ACTIVE", "createdAt": "2024-01-01T10:00:00", "totalAccess": 156, "todayAccess": 23 }

常见问题

短链接有有效期吗？
默认情况下，短链接的有效期为 365 天。过期后的链接将无法访问， 但您可以重新生成相同的短链接。
自定义短链有什么限制？
自定义短链只能包含字母、数字、下划线和连字符， 长度建议在 3-20 个字符之间。每个自定义短链必须是唯一的。
如何查看短链的访问统计？
您可以通过 API 接口 /api/stats/{shortUrl} 获取详细的访问统计信息， 包括总访问次数、今日访问次数、最近访问记录等。
短链接是否安全？
我们采用多重安全防护机制，包括恶意链接检测、访问频率限制等， 确保短链接的安全性。同时提供链接预览功能，让用户在访问前了解目标网站。
支持批量生成短链吗？
目前主要支持单个链接的生成。如果您有批量生成的需求， 可以通过 API 接口进行程序化调用。
如何删除或禁用短链？
目前通过管理界面可以管理短链的状态。您也可以联系我们的技术支持 来处理特殊情况下的链接管理需求。
